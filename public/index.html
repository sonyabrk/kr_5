API Генератора Случайных Цитат

Express.js REST API для управления коллекцией цитат

Статус: Загрузка...

Основные эндпоинты

GET Все цитаты
/api/quotes

Получить все цитаты. Поддерживает query-параметры:

limit - ограничить количество (например: ?limit=3)

author - фильтр по автору (например: ?author=джобс)

GET Случайная цитата
/api/quotes/random

Получить одну случайную цитату

GET Поиск цитат
/api/quotes/search?q=текст

Поиск по тексту или автору

GET Цитата по ID
/api/quotes/id

Получить конкретную цитату по её ID

POST Добавить цитату
/api/quotes

Добавить новую цитату (требует JSON тело)

{ "text": "Новая цитата", "author": "Автор" }
PUT Обновить цитату
/api/quotes/id

Обновить существующую цитату

DELETE Удалить цитату
/api/quotes/id

Удалить цитату по ID
