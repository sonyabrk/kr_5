# API Генератора Случайных Цитат
## Брюханова Софья ЭФБО-02-24 вариант 3
Express.js REST API для управления коллекцией цитат с полным CRUD функционалом.

## Функционал

- Получение всех цитат с пагинацией и фильтрацией
- Получение случайной цитаты
- Поиск по тексту и автору
- Добавление новых цитат
- Обновление существующих цитат
- Удаление цитат
- Кастомный middleware для логирования
- Раздача статических файлов
- Полная валидация данных
